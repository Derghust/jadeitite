# Jadeitite

Jadeitite is multifunctional and multiplatform for graphical and non-graphical application framework
written in C99 for windows and UNIX like operating systems.

Compatible with 64-bit and also 32-bit OS

## Compiler support

- GCC
- TCC
- CLang
